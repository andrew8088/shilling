# Budget Service

Monthly budget targets and comparison.

## Work
- Set/update budget target for an expense account + year/month
- Get budget vs actual for a given month (target vs sum of expense entries)
- Get budget summary across all budgeted expense accounts for a month
- Support rolling over unused budget (optional, flag-controlled)

## Acceptance Criteria
- Set and retrieve budget targets
- Budget vs actual computation tested with sample transactions
- Summary across multiple accounts tested
